# GU-BIOINFORMATICS-FINALPROJECT-SMM
Sameel, Mark, Maddie's Final Project

The goal of this project was to describe the gut microbiome of patients with ACVD and healthy controls. We acquired raw reads from another paper titled "The gut microbiome in atherosclerotic cardiovascular disease" (Zhuye et al. 2017). We then trimmed these raw reads using Trimmomatic, classified taxonomies present in the microbiomes using Kraken2, and analyzed the diversity of the samples using R.
